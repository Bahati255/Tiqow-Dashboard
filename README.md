1. **Clone the repo**  
   Or download the ZIP and extract:
   ```

   cd tiqow-dashboard
   ```

2. **Install dependencies**
   ```
   npm install
   ```

3. **Run the development server**
   ```
   npm run dev
   ```

4. **Visit**  
   Open [http://localhost:3000](http://localhost:3000) in your browser.
